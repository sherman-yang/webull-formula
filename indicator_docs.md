# Webull (微牛) 内置指标说明文档

本文档分为两大部分：自带指标与自定义指标。每个指标均包含：原理、详细使用说明与至少一个示例，便于直接上手。

## 自带指标 (Built-in Indicators)

### 1. 均线与趋势跟踪指标 (Moving Averages & Trend Following)

#### 指数移动平均线 (EMA)
*文件: `Exponential Moving Average.indicator`*
- **原理**: 对价格做指数加权平均，近期价格权重更高。该版本一次绘制多条 EMA（默认 5/10/20/30/60/120）。
- **使用说明**:
    - 观察短期 EMA 与长期 EMA 的交叉判断趋势变化。
    - 多头排列（短期在上、长期在下）为上升趋势；空头排列反之。
    - EMA 作为动态支撑/阻力位，配合回踩确认。
- **4H/24x7 推荐参数区间**:
    - `短期 EMA = 6-12`，`中期 EMA = 20-34`，`长期 EMA = 50-120`（保持递增）。
    - 参考组合: `8/21/55` 或 `10/30/90`。
- **例子**: 在 4H 级别上，EMA_1(5) 上穿 EMA_3(20)，且价格回踩 EMA_3 后反弹，可视为短线转强信号。

#### 位移均线 (DMA)
*文件: `Displaced Moving Average.indicator`*
- **原理**: 计算短期 SMA 与长期 SMA 的差值（DMA），再用 SMA 平滑得到 AMA。
- **使用说明**:
    - DMA 上穿 AMA 或零轴，动能由弱转强。
    - DMA 下穿 AMA 或零轴，动能转弱。
    - DMA 与价格的背离可提示拐点。
- **4H/24x7 推荐参数区间**:
    - `Short = 8-14`，`Long = 40-80`，`Multiplier = 6-12`。
- **例子**: DMA 在零轴下方收敛并上穿 AMA，提示下跌动能衰减，适合等待转强确认。

#### EMA 均线带 (EMA Ribbon)
*文件: `EMA Ribbon.indicator`*
- **原理**: 多条 EMA 形成“带状结构”，颜色反映斜率变化。
- **使用说明**:
    - 均线带向上扩散且价格位于带上方，趋势强。
    - 均线带收敛代表震荡，等待重新扩散方向。
    - 回踩带上沿不破，适合顺势加仓或持有。
- **4H/24x7 推荐参数区间**:
    - 起始长度 `6-10`，步长 `4-6`，末端长度 `50-80`。
- **例子**: 价格回踩短期 EMA 带后快速拉升，同时带状继续上扩，属于顺势信号。

#### 鳄鱼线 (Alligator)
*文件: `Alligator.indicator`*
- **原理**: 三条平滑均线（颚、齿、唇）带有位移，刻画趋势启动与休眠。
- **使用说明**:
    - 三线缠绕为“睡觉”，市场震荡。
    - 唇在上、齿居中、颚在下为多头排列，趋势向上。
    - 反向穿越或收敛预示趋势衰减。
- **例子**: 唇线上穿齿线且三线开始发散向上，常作为趋势启动信号。

#### TRIX
*文件: `TRIX.indicator`*
- **原理**: 对价格做三次 EMA 后计算变化率，过滤短期噪音。
- **使用说明**:
    - TRIX 上穿 0 轴代表趋势转多；下穿 0 轴代表转空。
    - 观察 TRIX 斜率变化判断动量强弱。
    - 可与均线趋势方向共振过滤假信号。
- **4H/24x7 推荐参数区间**:
    - `Length = 14-21`。
- **例子**: TRIX 从负值上穿 0 轴，同时价格突破前高，可视为趋势确认。

#### 平均趋向指数 (ADX)
*文件: `Average Directional Index.indicator`*
- **原理**: 衡量趋势强度，不区分方向。
- **使用说明**:
    - ADX 上升表示趋势在加强；下降表示趋势衰减。
    - 常用阈值 20/25 区分趋势与震荡。
    - ADX 高但走平，趋势可能进入尾声。
- **4H/24x7 推荐参数区间**:
    - `ADX Smoothing = 14-20`，`Length = 14-20`。
    - 强度阈值参考 `20-25`。
- **例子**: ADX 从 18 上升到 25 以上，说明行情由震荡转为趋势。

#### 方向性运动指数 (DMI)
*文件: `Directional Movement Index.indicator`*
- **原理**: +DI 与 -DI 表示方向强弱，ADX 表示趋势强度。
- **使用说明**:
    - +DI 上穿 -DI 代表多头占优；反之空头占优。
    - 结合 ADX > 20/25 过滤弱势交叉。
    - DI 交叉频繁且 ADX 低，通常为震荡。
- **4H/24x7 推荐参数区间**:
    - `DI Length = 14-20`，`ADX Smoothing = 14-20`。
- **例子**: +DI 上穿 -DI 且 ADX > 25，可视为顺势做多条件之一。

#### ADX 与 DI 组合
*文件: `ADX and DI.indicator`*
- **原理**: 同时绘制 DI+、DI- 与 ADX，并提供阈值参考线。
- **使用说明**:
    - DI 交叉给方向；ADX 突破阈值线表示趋势足够强。
    - DI 交叉但 ADX 低于阈值时谨慎对待。
    - ADX 高位回落时，趋势可能进入末期。
- **4H/24x7 推荐参数区间**:
    - `len = 14-20`，`th = 20-25`。
- **例子**: DI+ 上穿 DI-，但 ADX 仍低于 20，先观望等待强度确认。

#### 涡旋指标 (Vortex Indicator)
*文件: `Vortex Indicator.indicator`*
- **原理**: 通过高低点关系计算 VI+ 与 VI-，判断趋势方向。
- **使用说明**:
    - VI+ 上穿 VI- 代表上升趋势启动。
    - VI- 上穿 VI+ 代表下降趋势启动。
    - 线条持续发散表示趋势增强。
- **4H/24x7 推荐参数区间**:
    - `Period = 10-21`。
- **例子**: VI+ 上穿 VI- 且持续拉开，为上升趋势确认。

#### 趋势魔力 (Trend Magic)
*文件: `Trend Magic.indicator`*
- **原理**: 基于 CCI 与 ATR 动态生成趋势跟踪线，用作跟踪止损。
- **使用说明**:
    - 价格在 MagicTrend 线上方且线向上，多头趋势。
    - 价格跌破 MagicTrend 线可视为趋势结束/止损信号。
    - 可作为持仓的移动止损线。
- **4H/24x7 推荐参数区间**:
    - `CCI period = 14-30`，`ATR Period = 5-14`，`ATR Multiplier = 1.0-2.0`。
- **例子**: 多头持仓时以 MagicTrend 线作为止损位，线不破则继续持有。

### 2. 动量与震荡指标 (Momentum & Oscillators)

#### MACD
*文件: `Moving Average Convergence Divergence.indicator`*
- **原理**: 快慢 EMA 差值为 MACD 线，信号线为其 EMA，柱状图显示差值。
- **使用说明**:
    - MACD 上穿信号线为多头动能增强。
    - MACD 下穿信号线为空头动能增强。
    - 柱状图由负转正或由正转负反映动能变化。
- **4H/24x7 推荐参数区间**:
    - `Fast = 10-14`，`Slow = 24-30`，`Signal = 7-10`。
- **例子**: MACD 线在零轴下方金叉，同时柱状图开始放大，视为反转预警。

#### MACD（带交叉标记）
*文件: `MACD with crossing signal.indicator`*
- **原理**: 与标准 MACD 相同，额外用圆点标记交叉位置。
- **使用说明**:
    - 绿点出现代表 MACD 上穿信号线。
    - 红点出现代表 MACD 下穿信号线。
    - 结合柱状图方向判断动能强弱。
- **4H/24x7 推荐参数区间**:
    - `Fast = 10-14`，`Slow = 24-30`，`Signal = 7-10`。
- **例子**: 红点出现且柱状图转负，表示空头动能开始占优。

#### 动量指标 (Momentum)
*文件: `Momentum.indicator`*
- **原理**: 当前价格与 N 周期前价格的差值。
- **使用说明**:
    - 指标上穿 0 轴代表动量由弱转强。
    - 连续抬高代表上升动能增强。
    - 与价格背离可提示趋势拐点。
- **4H/24x7 推荐参数区间**:
    - `Length = 8-20`。
- **例子**: MOM 上穿 0 轴且持续走高，配合价格突破平台视为趋势启动。

#### 动量震荡指标 (Awesome Oscillator, AO)
*文件: `Awesome Oscillator.indicator`*
- **原理**: 5 与 34 周期 SMA（基于中间价）的差值。
- **使用说明**:
    - AO 上穿 0 轴为多头动能增强。
    - 柱状图颜色由红转绿，动量开始改善。
    - 可观察“双峰背离”信号。
- **例子**: AO 在零轴下方出现更高的低点并转绿，提示下跌动能减弱。

#### 艾略特波浪震荡指标 (EWO)
*文件: `Elliott Wave Oscillator.indicator`*
- **原理**: 5 与 35 周期 EMA 的差值，以柱状图显示。
- **使用说明**:
    - EWO 上穿 0 轴表示短期强于长期。
    - 柱状图逐步放大说明动能增强。
    - 结合价格结构判断浪形变化。
- **例子**: EWO 由负转正且价格突破下降趋势线，常视作上升浪开始。

#### 钱德动量摆动指标 (CMO)
*文件: `Chande Momentum Oscillator.indicator`*
- **原理**: 上涨与下跌动量的对比，围绕 0 轴波动。
- **使用说明**:
    - CMO 上穿 0 轴代表动量转强。
    - 常用 ±50 作为强弱区分参考。
    - 与价格背离可提示反转。
- **例子**: CMO 从 -60 回升至 0 附近，显示空头动能显著衰减。

#### 相对强弱指数 (RSI)
*文件: `Relative Strength Index.indicator`*
- **原理**: 衡量价格上涨与下跌幅度的相对强弱，范围 0-100。
- **使用说明**:
    - RSI > 70 常视为超买，< 30 常视为超卖。
    - RSI 回到 50 上方通常代表多头占优。
    - RSI 与价格背离可提示反转。
- **4H/24x7 推荐参数区间**:
    - `RSI Length = 18-24`。
    - `Overbought = 70-75`，`Oversold = 25-30`。
- **例子**: RSI 跌破 30 后迅速回到 30 上方，同时价格止跌，属于反弹信号。

#### RSI 趋势版
*文件: `RSI Trend.indicator`*
- **原理**: 使用 60/40 作为趋势区间，颜色反映趋势状态。
- **使用说明**:
    - RSI 高于 60 表示趋势偏多。
    - RSI 低于 40 表示趋势偏空。
    - 中间区域代表震荡或趋势过渡。
- **4H/24x7 推荐参数区间**:
    - `RSI Length = 14-21`，`Upper = 55-65`，`Lower = 35-45`。
- **例子**: RSI 长期维持在 60 上方且颜色为绿，趋势偏多。

#### RSI 阈值跟踪
*文件: `RSI Threshold Tracker.indicator`*
- **原理**: RSI 结合自身的布林带，形成动态阈值区间。
- **使用说明**:
    - RSI 突破上动态带，动量过热。
    - RSI 跌破下动态带，动量过弱。
    - 中间带为“中性区”，过滤噪音。
- **4H/24x7 推荐参数区间**:
    - `RSI_period = 14-21`，`Basis_BB = 20-34`，`Stdev = 2`，`Dispersion = 0.08-0.15`。
- **例子**: RSI 从中性区上穿上动态带后，价格出现加速拉升。

#### 随机指标 (Stochastic)
*文件: `Stochastic.indicator`*
- **原理**: 比较收盘价与一定周期内高低区间，输出 %K 与 %D。
- **使用说明**:
    - %K 上穿 %D 为多头信号。
    - 超买区 > 80，超卖区 < 20。
    - 在震荡区间效果更好。
- **4H/24x7 推荐参数区间**:
    - `K Period = 14-21`，`D Period = 3-5`，`K Smooth = 3`。
- **例子**: %K 在 20 下方上穿 %D，提示短线反弹机会。

#### 快速随机指标 (Fast Stochastic)
*文件: `Fast Stochastic Oscillator.indicator`*
- **原理**: 未做额外平滑的随机指标，反应更敏感。
- **使用说明**:
    - 更适合短周期或震荡行情。
    - 超买/超卖信号更频繁，需结合趋势过滤。
    - %K/%D 交叉仍为主要信号。
- **4H/24x7 推荐参数区间**:
    - `%K Length = 10-14`，`%D Length = 3`。
- **例子**: 在区间震荡中，%K 于 80 上方下穿 %D 作为短线回落提示。

#### 随机 RSI (Stochastic RSI)
*文件: `Stochastic RSI.indicator`*
- **原理**: 先计算 RSI，再对 RSI 进行随机化处理。
- **使用说明**:
    - > 80 超买，< 20 超卖。
    - 适合捕捉更短周期的动量变化。
    - %K/%D 交叉为主要参考。
- **4H/24x7 推荐参数区间**:
    - `RSI Length = 14-21`，`Stoch Length = 14-21`，`K/D Smooth = 3-5`。
- **例子**: Stoch RSI 在 20 下方金叉并向上突破，提示短线回升。

#### 随机动量指数 (SMI)
*文件: `Stochastic Momentum Index.indicator`*
- **原理**: 该实现以 TSI（双平滑动量）作为主线，并绘制信号线与振荡柱。
- **使用说明**:
    - 主线（Indicator）上穿信号线为动量转强。
    - 柱状图由负转正表示动能改善。
    - 与价格背离可提示拐点。
- **4H/24x7 推荐参数区间**:
    - `Short = 5-8`，`Long = 20-30`，`Signal = 5-8`。
- **例子**: Indicator 上穿 Signal 且柱状图转正，提示反弹开始。

#### 商品通道指数 (CCI)
*文件: `Commodity Channel Index.indicator`*
- **原理**: 价格与其均值的偏离程度标准化。
- **使用说明**:
    - CCI > +100 表示强势或超买。
    - CCI < -100 表示弱势或超卖。
    - 结合趋势方向过滤逆势信号。
- **4H/24x7 推荐参数区间**:
    - `Length = 20-30`。
- **例子**: CCI 从 -120 上穿 -100，提示下跌动能减弱。

#### 成交量加权 CCI
*文件: `CCI with volume weighted.indicator`*
- **原理**: 以成交量加权价格计算 CCI，提高量价结合度。
- **使用说明**:
    - 仍以 ±100 作为关键阈值。
    - 指标上会标记穿越 ±100 的圆点。
    - 适合观察放量趋势的强弱变化。
- **4H/24x7 推荐参数区间**:
    - `Length = 14-20`。
- **例子**: CCI 放量上穿 +100 并出现绿点，提示趋势走强。

#### 威廉指标 (Williams %R)
*文件: `Williams %R.indicator`*
- **原理**: 衡量收盘价在最近区间中的相对位置，范围 -100 到 0。
- **使用说明**:
    - > -20 为超买区，< -80 为超卖区。
    - 由超卖区回到 -50 上方常视作反弹。
    - 在强趋势中易钝化。
- **4H/24x7 推荐参数区间**:
    - `Length = 14-21`。
- **例子**: %R 从 -85 回升并上穿 -80，提示短线反弹。

#### 乖离率 (Kairi Relative Index, KRI)
*文件: `Kairi Relative Index.indicator`*
- **原理**: 价格相对均线的百分比偏离。
- **使用说明**:
    - KRI > 0 表示价格高于均线，< 0 反之。
    - 常以 ±5%/±10% 作为偏离强度参考。
    - 用于判断均值回归或过热。
- **4H/24x7 推荐参数区间**:
    - `Length = 30-60`。
- **例子**: KRI 由 +8% 回落并接近 0，提示上行乖离回归。

#### 波浪趋势 (Wave Trend)
*文件: `Wave Trend.indicator`*
- **原理**: 以典型价格与 EMA 偏离计算 WT1/WT2，两线交叉与超买超卖区结合使用。
- **使用说明**:
    - WT1 上穿 WT2 为多头信号，下穿为弱势信号。
    - 超买区（如 60 附近）出现死叉，常见回调。
    - 超卖区（如 -60 附近）出现金叉，常见反弹。
- **4H/24x7 推荐参数区间**:
    - `Channel Length = 9-14`，`Average Length = 18-28`。
    - `OB = 55-65`，`OS = -55 至 -65`。
- **例子**: WT1 在 -60 附近上穿 WT2，提示短线反弹开始。

### 3. 波动率与区间指标 (Volatility & Bands)

#### 布林带 (Bollinger Bands)
*文件: `Bollinger Bands.indicator`*
- **原理**: 中轨为 SMA，上下轨为标准差通道。
- **使用说明**:
    - 价格触及上轨常被视为超买，触及下轨为超卖。
    - 带宽收窄（挤压）预示可能突破。
    - 沿上轨运行常代表强势趋势。
- **4H/24x7 推荐参数区间**:
    - `Length = 20-30`，`StdDev = 1.8-2.2`。
- **例子**: 布林带明显收窄后，价格放量突破上轨，提示趋势启动。

#### 布林带宽度 (Bollinger Bands Width)
*文件: `Bollinger Bands Width.indicator`*
- **原理**: 量化上下轨距离，反映波动率。
- **使用说明**:
    - BBW 处于低位说明市场震荡且可能临近突破。
    - BBW 快速上升说明波动率扩张。
    - 可与价格突破方向配合使用。
- **4H/24x7 推荐参数区间**:
    - `Length = 20-30`，`StdDev = 1.8-2.2`。
- **例子**: BBW 长期低位后突然抬升，价格同步突破箱体。

#### Percent B
*文件: `Percent B.indicator`*
- **原理**: 衡量价格在布林带中的相对位置（0-1 之间，超出则 <0 或 >1）。
- **使用说明**:
    - %B > 1 代表价格突破上轨。
    - %B < 0 代表价格跌破下轨。
    - 常用 0.5 作为中轴参考。
- **4H/24x7 推荐参数区间**:
    - `Length = 20-30`，`StdDev = 1.8-2.2`。
- **例子**: %B 从 0.2 上升并突破 0.8，说明价格从偏弱转强。

#### 斐波那契布林带 (Fibonacci Bollinger Bands)
*文件: `Fibonacci Bollinger Bands.indicator`*
- **原理**: 以 VWMA 为中线，使用标准差并按斐波那契比例划分多层通道。
- **使用说明**:
    - 不同层级可作为分批止盈/加仓参考。
    - 价格多次在某一层级附近反应，说明该层级有效。
    - 适合趋势中分层管理仓位。
- **4H/24x7 推荐参数区间**:
    - `Length = 150-240`，`Multiplier = 2.5-3.5`。
- **例子**: 上涨趋势中价格多次在 0.618 带附近获得支撑，可作为回踩买点参考。

#### 标准差 (Standard Deviation)
*文件: `Standard Deviation.indicator`*
- **原理**: 统计价格在一定周期内的波动幅度。
- **使用说明**:
    - 标准差升高代表波动率扩张。
    - 标准差降低代表波动率收缩。
    - 与价格结构结合判断突破风险。
- **4H/24x7 推荐参数区间**:
    - `Length = 20-30`。
- **例子**: 标准差从低位快速抬升且价格突破区间上沿，提示波动扩张。

#### 肯特纳通道 (Keltner Channels)
*文件: `Keltner Channels.indicator`*
- **原理**: EMA 为中线，ATR 作为通道宽度。
- **使用说明**:
    - 价格突破上轨常视作趋势延续信号。
    - 价格回踩中轨不破，多头仍强。
    - 与布林带配合可识别挤压与突破。
- **4H/24x7 推荐参数区间**:
    - `Length = 20-30`，`Multiplier = 1.0-2.0`，`Exponential = true`。
- **例子**: 价格突破上轨并站稳，视作趋势突破确认。

#### 斐波那契区间 (Fibonacci Levels)
*文件: `Fibonacci Levels.indicator`*
- **原理**: 以 EMA 与标准差为基础生成动态斐波那契区间线。
- **使用说明**:
    - 动态区间线可用作支撑/阻力参考。
    - 多次回踩某条区间线不破，说明该线有效。
    - 适合趋势中分批止盈或回撤管理。
- **4H/24x7 推荐参数区间**:
    - `100 MA = 80-120`。
- **例子**: 价格在 UP 2 / UP 3 附近遇阻并回落，可作为减仓参考。

### 4. 成交量与资金流指标 (Volume & Flow)

#### 能量潮 (OBV)
*文件: `On-Balance Volume.indicator`*
- **原理**: 按涨跌方向累计成交量。
- **使用说明**:
    - OBV 上升代表资金持续流入。
    - 价格创新高而 OBV 未创新高提示背离。
    - 适合用于趋势确认。
- **例子**: 价格上涨同时 OBV 创新高，趋势更可信。

#### 资金流量指数 (MFI)
*文件: `Money Flow Index.indicator`*
- **原理**: 成交量加权的 RSI，范围 0-100。
- **使用说明**:
    - MFI > 80 为超买，< 20 为超卖。
    - 与价格背离常提示趋势反转。
    - 适合与 RSI 互相验证。
- **4H/24x7 推荐参数区间**:
    - `Length = 14-21`。
- **例子**: MFI 从 15 回升至 30，同时价格止跌，提示资金回流。

#### 价量趋势 (PVT)
*文件: `Price Volume Trend.indicator`*
- **原理**: 按价格涨跌幅度加权的累计成交量。
- **使用说明**:
    - PVT 上穿信号线表示资金偏多。
    - PVT 走平而价格上涨，可能是虚涨。
    - 趋势中 PVT 作为确认指标。
- **4H/24x7 推荐参数区间**:
    - `Signal Length = 13-21`，`Signal Type = EMA/SMA`。
- **例子**: PVT 上穿信号线且价格突破区间上沿，作为趋势确认信号。

#### 成交量流量指标 (VFI)
*文件: `Volume Flow.indicator`*
- **原理**: 通过成交量与价格变化过滤异常波动，衡量真实资金流向。
- **使用说明**:
    - VFI 上穿 0 轴表示资金流入增强。
    - VFI 下穿 0 轴表示资金流出。
    - 柱状图可观察短期动能变化。
- **4H/24x7 推荐参数区间**:
    - `VFI length = 100-150`，`coef = 0.1-0.3`，`Max. vol. cutoff = 2.0-3.0`，`signal = 3-7`。
- **例子**: VFI 上穿 0 轴并持续上升，配合价格突破可确认趋势。

#### 艾尔德强力指数 (EFI)
*文件: `Elder's Force Index.indicator`*
- **原理**: 价格变化与成交量的乘积，再做 EMA 平滑。
- **使用说明**:
    - EFI 上穿 0 轴为多头力量增强。
    - EFI 大幅放大表示强趋势。
    - 适合判断趋势加速或衰减。
- **4H/24x7 推荐参数区间**:
    - `Length = 13-21`。
- **例子**: EFI 从负转正并持续抬高，提示多头力量回归。

#### 蔡金震荡指标 (Chaikin Oscillator)
*文件: `Chaikin Oscillator.indicator`*
- **原理**: 对 A/D 线做快慢 EMA 差值，反映资金流变化。
- **使用说明**:
    - 震荡指标上穿 0 轴为资金流入。
    - 下穿 0 轴为资金流出。
    - 适合与价格趋势配合确认。
- **4H/24x7 推荐参数区间**:
    - `Fast = 3-5`，`Slow = 10-20`。
- **例子**: Chaikin Osc 上穿 0 轴且价格突破短期阻力，提示趋势上行。

#### 威廉累积/派发 (WVAD)
*文件: `Williams Accumulation-Distribution.indicator`*
- **原理**: 通过收盘价与开盘价的差值衡量当根 K 线内的买卖力量，并累积。
- **使用说明**:
    - WVAD 上升说明买方占优，下降说明卖方占优。
    - 与价格背离可提示趋势变化。
    - EMA 线用于平滑观察趋势。
- **4H/24x7 推荐参数区间**:
    - `ema length = 13-21`。
- **例子**: 价格横盘但 WVAD 持续上升，说明资金在吸筹。

### 5. 其他 / 专用指标

#### 瓦达·阿塔爆炸 (Waddah Attar Explosion)
*文件: `Waddah Attar Explosion.indicator`*
- **原理**: 结合 MACD 动量与布林带宽度，比较趋势动能与“爆炸线/死区线”。
- **使用说明**:
    - 绿色/红色柱体代表趋势动能方向。
    - 当趋势柱体高于 Explosion Line 且高于 Dead Zone，趋势信号更可靠。
    - 适合捕捉爆发行情。
- **4H/24x7 推荐参数区间**:
    - `Sensitivity = 120-200`，`FastEMA = 15-25`，`SlowEMA = 30-50`。
    - `BB Channel Length = 20-30`，`BB Stdev = 2.0-2.5`。
- **例子**: 绿色柱体突破 Explosion Line 且高于 Dead Zone，可视为强势突破信号。

### 神奇九转完整版 (TD Sequential Complete)
*文件: `Magic 9 Turn.indicator`*
- **原理**: 通过 Setup(9) 与 Countdown(13) 计数识别趋势衰竭与反转风险，包含 Perfected、Aggressive、Combo 等强化条件，并绘制 TDST 支撑/阻力线。
- **使用说明**:
    - **Setup 9**: 连续 9 根满足条件的 K 线计数完成，提示趋势可能暂停。
    - **Perfected**: Setup 9 且第 8/9 根极值突破第 6/7 根，提高信号质量。
    - **Countdown 13**: Setup 完成后开始计数，达到 13 表示趋势耗尽概率增大。
    - **Aggressive/Combo**: 更激进或更严格的计数版本，提前或加速给出信号。
    - **TDST 线**: Setup 完成后绘制的支撑/阻力，用于趋势延续/破位判断。
- **信号触发条件**:

| 信号 | 触发条件 | 说明 |
|------|----------|------|
| **Buy 9** | 连续 9 根 Close < Close[4] | 下跌趋势可能暂停 |
| **Sell 9** | 连续 9 根 Close > Close[4] | 上涨趋势可能暂停 |
| **Buy Perfected** | Buy 9 + 第 8/9 根低点 ≤ 第 6/7 根低点 | 高质量信号 |
| **Sell Perfected** | Sell 9 + 第 8/9 根高点 ≥ 第 6/7 根高点 | 高质量信号 |
| **Buy 13** | Buy 9 后累计 13 次 Close ≤ Low[2] | 趋势耗尽 |
| **Sell 13** | Sell 9 后累计 13 次 Close ≥ High[2] | 趋势耗尽 |
| **Aggressive** | Countdown 8-12 时满足 Close ≤ Low[1-3] / Close ≥ High[1-3] | 提前入场 |
| **Combo** | Close < Low[2] / Close > High[2] | 更严格, 出信号更快 |

- **颜色+大小规则（优先级越高, 颜色越深, 标记越大）**:
    - 买入: 浅绿小圆 `#90EE90` (Combo) → 草绿小叉 `#32CD32` (Aggressive) → 绿色中圆 `#00AA00` (Buy 9) → 深绿大圆 `#006600` (Buy Perfected) → 墨绿大叉 `#003300` (Buy 13)。
    - 卖出: 浅红 → 粉红 → 红色 → 深红 → 暗红 (对应同等优先级)。
- **注意**: 下跌趋势中 Sell 信号较多, 上涨趋势中 Buy 信号较多。
- **适用范围**: 日线/周线效果最佳, 4H 可作为辅助确认。
- **时间框架参数建议(连续交易市场)**:

| 时间框架 | Lookback | 说明 |
|---------|---------|------|
| 4H | 4 | 标准设定 |
| 3m | 5 | 降噪 |
| 2m | 5 | 降噪 |
| 1m | 6 | 进一步降噪 |

- **例子**: 出现 Sell 9 后又满足 Sell Perfected, 同时价格触及 TDST 阻力线附近, 可视为趋势进入尾声的预警。

### 标准分数 (Z-Score)
*文件: `Z-Score.indicator`*
- **原理**: 用标准差衡量价格偏离均值的程度，`Z = (价格 - 均值) / 标准差`。
- **使用说明**:
    - Z > +2 可能超买，Z < -2 可能超卖。
    - Z 长期 > 0 说明价格高于均值，< 0 说明偏弱。
    - -1 到 +1 可视为“正常波动区”。
    - 穿越 ±2σ 会显示信号圆点（可关闭）。
- **4H/24x7 推荐参数区间**:
    - `Lookback = 36-60`，`Band 1σ = 0.8-1.2`，`Band 2σ = 1.8-2.5`，`Signal = 3-7`。
- **时间框架参数建议(连续交易市场)**:

| 时间框架 | Lookback | Band 1σ | Band 2σ | Signal |
|---------|---------|---------|---------|--------|
| 4H | 42 | 1.0 | 2.0 | 5 |
| 3m | 120 | 1.0 | 2.2 | 7 |
| 2m | 150 | 1.0 | 2.3 | 7 |
| 1m | 200 | 1.2 | 2.5 | 9 |

- **例子**: Z-score 低于 -2 后快速回到 -1 以上，提示均值回归启动。

### 标准分数（收益率版，Z-Score Return）
*文件: `Z-Score (Return).indicator`*
- **原理**: 先将价格转换为对数收益率，再对收益率计算 Z-Score，适合高波动品种。
- **使用说明**:

    - 穿越 ±2σ 会显示信号圆点（可关闭）。

- **4H/24x7 推荐参数区间**:
    - `Lookback = 36-60`，`Band 1σ = 0.8-1.2`，`Band 2σ = 1.8-2.5`，`Signal = 3-7`。
- **时间框架参数建议(连续交易市场)**:

| 时间框架 | Lookback | Band 1σ | Band 2σ | Signal |
|---------|---------|---------|---------|--------|

- **例子**: 高波动品种中，收益率 Z-score 从 -2 下方回到 -1，提示短期回归。

### RSI（4H 优化）
